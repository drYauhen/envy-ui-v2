# generated/tsx/

Purpose: TypeScript/TSX platform artifacts derived from tokens.
Source/workflow: `scripts/generate-contracts.ts` reads token JSON directly.
Consumers/destinations: TSX component implementations and stories.
Artifact types/roles: Component contracts (literal unions and const arrays).

Producers
- `scripts/generate-contracts.ts`

Expected artifacts
- `button.contract.ts` (token-derived component axes for Button)

Regeneration
- `node scripts/generate-contracts.ts`
