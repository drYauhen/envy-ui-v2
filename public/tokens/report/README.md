# Report Context

Report context defines tokens for printable and exportable documents.

## Overview

The Report context is optimized for print and screen outputs with:
- Print-first design approach
- Optimized typography for readability
- High contrast for print media

## Structure

- **Foundations**: Base tokens (currently empty, structure in place)
- **Semantic**: Semantic tokens (currently empty, structure in place)
- **Components**: Component-specific tokens (currently empty, structure in place)
- **Themes**: Output-specific themes (print, screen)

## Status

This context is planned for future development. The structure is in place to support report-specific tokens (optimized for print and screen outputs) when needed.
