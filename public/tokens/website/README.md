# Website Context

Website context defines tokens for CMS and content websites.

## Overview

The Website context is optimized for content-focused websites with:
- Content-first design approach
- Relaxed spacing and typography
- Marketing and editorial layouts

## Structure

- **Foundations**: Base tokens (currently empty, structure in place)
- **Semantic**: Semantic tokens (currently empty, structure in place)
- **Components**: Component-specific tokens (currently empty, structure in place)
- **Themes**: Visual themes (default, dark)

## Status

This context is planned for future development. The structure is in place to support website-specific tokens when needed.
