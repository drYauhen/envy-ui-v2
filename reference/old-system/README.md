# Old System Component References

Place references to old system components here.

## Format

For each component, create a directory with:
- `component.html` or `component.css` - Original implementation
- `NOTES.md` - Key features, behaviors, and migration considerations

## Example Structure

```
old-system/
  sidebar/
    sidebar.html
    sidebar.css
    NOTES.md
  table/
    table.html
    table.css
    NOTES.md
```

## Referencing External Repository

To reference components from another local repository, you can:

1. Copy files here for easy reference
2. Or provide absolute path when asking to analyze:
   `/path/to/old-repo/components/sidebar/`
